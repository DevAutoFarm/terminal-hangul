# Contributing to terminalHangul

terminalHangul에 기여해 주셔서 감사합니다!

## How to Contribute

### Reporting Bugs

버그를 발견하셨나요? [GitHub Issues](https://github.com/USERNAME/terminal-hangul/issues)에 제보해 주세요.

버그 리포트에 포함해야 할 정보:
- macOS 버전
- 사용 중인 터미널 앱 (Terminal.app, iTerm2, etc.)
- 사용 중인 한글 입력기 (macOS 기본, Gureum, etc.)
- 문제 재현 단계
- 예상 동작 vs 실제 동작

### Suggesting Features

새로운 기능을 제안하고 싶으시다면 Issue를 열어주세요. 제안 시 다음을 포함해 주세요:
- 기능 설명
- 사용 사례 (왜 이 기능이 필요한지)
- 가능하다면 구현 아이디어

### Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Development Setup

```bash
# Clone the repository
git clone https://github.com/USERNAME/terminal-hangul.git
cd terminal-hangul

# Build
swift build

# Run
swift run
```

### Code Style

- Swift 표준 코딩 스타일을 따릅니다
- 의미 있는 커밋 메시지를 작성해 주세요
- 새로운 기능에는 적절한 주석을 추가해 주세요

## Code of Conduct

- 서로를 존중해 주세요
- 건설적인 피드백을 제공해 주세요
- 다양한 의견을 환영합니다

## Questions?

질문이 있으시면 [GitHub Discussions](https://github.com/USERNAME/terminal-hangul/discussions)를 이용해 주세요.
